Security first

Designed for workflows where mistakes and tampering matter.

Schemarift Postgres is built around package integrity, credential care, manual approval, and visible deployment outcomes.

Package integrity

  • Signed migration manifests
  • SHA-256 file hashes
  • Rejects missing, duplicate, or tampered files

Credential care

  • Connection passwords are not stored
  • Logs redact sensitive values
  • DPAPI protects local settings where used

Production control

  • Target must be analyzed before execution
  • Risky changes are blocked for review
  • Production confirmation is required
